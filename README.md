# SOAR_Playbook
Automatisation d'un playbook SOC avec assistance IA

## Objectif
Le projet démontre l'utilisation d'un playbook automatisé avec une assistance IA, qui est désigné pour aider les analyses d'alertes (SOC) au triage et au gestion des alertes.

## Travail
Le travail consiste à créer une alerte imaginaire avec des champs écrit comme une vraie alerte SOC, puis parser cette alerte selon les champs, ajoute un score de priorité pour chaques alertes générées, gestion de décision, et créer un rapport. J'ajoute aussi de l'assistance IA pour chaques étapes et facilite ainsi l'analyse d'alerte.

## Comment exécuter?
```bash
python3 src/playbook.py
```
